# India Climate Analytics Dashboard

## Overview
This project analyzes climate patterns across major Indian cities during 2024–2025 using Python-based data analytics and visualization techniques.

## Dataset
- 7,310 records
- 13 climate attributes
- 10 major Indian cities

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
- Kolkata recorded the highest average maximum temperature.
- Mumbai showed the cleanest air quality based on AQI.
- Lucknow had the highest average cloud cover.
- Positive correlation observed between rainfall and humidity.
